## Visão Geral

O Tilemap Editor é uma ferramenta robusta para o desenho de níveis 2D baseados em grelha (grid). Ele permite que os designers utilizem "tilesets" para pintar cenários complexos de forma eficiente. A ferramenta suporta múltiplas camadas para permitir a separação entre elementos de fundo, elementos interativos e dados de colisão.

Além das ferramentas básicas de pintura, o editor inclui capacidades de "Auto-tiling", que aplicam automaticamente as regras de vizinhança para escolher o tile correto (ex: cantos, bordas de grama/terra). Isto reduz drasticamente o tempo necessário para criar terrenos orgânicos. A integração com o sistema de assets do Caffeine permite a exportação direta para ficheiros `.caf` otimizados para o runtime.

## Implementação

O editor de tilemaps gere uma estrutura de dados composta por camadas de células, onde cada célula armazena um índice de tile e flags de configuração.

### Estrutura de Dados do Tilemap

```cpp
namespace Caffeine::Editor {

struct TileCell {
    int tileID = -1;
    uint32_t flags = 0; // Ex: Inverter X, Inverter Y, Rotação
};

class TileLayer {
public:
    std::string m_name;
    std::vector<TileCell> m_cells;
    int m_width, m_height;
    bool m_isVisible = true;
    bool m_isLocked = false;

    TileCell& GetCell(int x, int y) { return m_cells[y * m_width + x]; }
};

class Tilemap {
public:
    std::vector<TileLayer> m_layers;
    Ref<Tileset> m_activeTileset;
    float m_tileSize = 32.0f;

    void AddLayer(const std::string& name);
    void RemoveLayer(int index);
};

class TilemapEditorPanel : public EditorPanel {
public:
    TilemapEditorPanel();
    void OnImGuiRender() override;
    void OnSceneRender(const SceneView& view);

private:
    void RenderPalette();
    void RenderToolbar();
    void PaintTile(int layerIdx, int x, int y, int tileID);
    void FloodFill(int layerIdx, int x, int y, int targetID, int replacementID);

    Tilemap m_currentMap;
    int m_selectedTileID = 0;
    int m_currentLayer = 0;
    ToolMode m_currentTool = ToolMode::Brush;
};

} // namespace Caffeine::Editor
```

### Algoritmo de Auto-tiling (Pseudo-C++)

```cpp
void TilemapEditorPanel::ApplyAutoTile(int layerIdx, int x, int y) {
    auto& layer = m_currentMap.m_layers[layerIdx];
    
    // Máscara de 8 bits para vizinhos (3x3 à volta do centro)
    uint8_t neighborMask = 0;
    if (HasNeighbor(layer, x, y-1)) neighborMask |= (1 << 0); // Norte
    if (HasNeighbor(layer, x+1, y)) neighborMask |= (1 << 1); // Este
    if (HasNeighbor(layer, x, y+1)) neighborMask |= (1 << 2); // Sul
    if (HasNeighbor(layer, x-1, y)) neighborMask |= (1 << 3); // Oeste
    // ... incluir diagonais se necessário

    // Mapear máscara para o ID do tile correto definido nas regras do Tileset
    int correctTileID = m_currentTileset->GetRuleMatch(neighborMask);
    layer.GetCell(x, y).tileID = correctTileID;
}
```

## Decisões de Design

| Decisão | Justificativa |
|---------|---------------|
| Armazenamento por Camadas | Permite profundidade visual e separação lógica (ex: camada de colisão invisível). |
| Sistema de Auto-tiling | Aumenta a produtividade ao automatizar a escolha de tiles de transição. |
| Ferramenta Bucket Fill | Essencial para preencher grandes áreas rapidamente (ex: fundos ou oceanos). |
| Grid Snapping Forçado | Garante que os tiles estão sempre perfeitamente alinhados, evitando erros visuais. |
