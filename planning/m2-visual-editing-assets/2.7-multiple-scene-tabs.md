## Visão Geral

O sistema de Multiple Scene Tabs permite que o utilizador trabalhe em várias cenas simultaneamente dentro do mesmo editor. Cada tab (separador) representa uma cena distinta com o seu próprio `ECS World` independente, permitindo comparar níveis, copiar entidades entre cenas ou manter uma cena de teste aberta enquanto se edita a cena principal.

Esta funcionalidade aproxima o Caffeine Studio de ferramentas profissionais de produtividade, permitindo uma gestão de janelas flexível onde os separadores podem ser reordenados ou destacados para janelas flutuantes independentes.

---

## Implementação

O `SceneTabManager` gere uma coleção de instâncias de `Scene`, cada uma associada a um separador no ImGui.

### Estrutura de Classes

```cpp
namespace Caffeine::Editor {

    struct SceneTab {
        std::string Name;
        std::shared_ptr<Scene> SceneInstance;
        bool IsDirty = false;
        bool IsActive = false;
    };

    class SceneTabManager {
    public:
        void OnImGuiRender();
        
        void OpenScene(const std::filesystem::path& path);
        void CloseScene(int index);
        void SetActiveScene(int index);

        Scene* GetActiveScene() { return m_Tabs[m_ActiveTabIndex].SceneInstance.get(); }

    private:
        void RenderTabBar();
        void HandleTabShortcuts();

        std::vector<SceneTab> m_Tabs;
        int m_ActiveTabIndex = 0;
    };
}
```

### Isolamento de ECS World

Cada cena aberta possui o seu próprio gestor de entidades e sistemas.

```cpp
void SceneTabManager::OpenScene(const std::filesystem::path& path) {
    auto scene = std::make_shared<Scene>();
    scene->Load(path);
    
    SceneTab tab;
    tab.Name = path.filename().string();
    tab.SceneInstance = scene;
    m_Tabs.push_back(tab);
    
    m_ActiveTabIndex = m_Tabs.size() - 1;
}
```

### Funcionalidade de Destacamento (Floating)

Utilizando a funcionalidade de Viewports do ImGui, os separadores podem ser arrastados para fora da janela principal.

```cpp
if (ImGui::BeginTabItem(tab.Name.c_str())) {
    // Renderizar Viewport para esta cena específica
    m_Viewport.Render(tab.SceneInstance);
    ImGui::EndTabItem();
}
```

## Decisões de Design

| Decisão | Justificativa |
|---------|---------------|
| ECS World por Tab | Garante que as alterações numa cena não afetem as outras, evitando corrupção de dados entre níveis. |
| Indicador "Dirty" (*) | Mostrar um asterisco no nome do separador alerta o utilizador para alterações não guardadas antes de fechar. |
| Drag-to-Reorder | Segue as convenções de navegadores web e IDEs (como VS Code) para uma experiência familiar. |

---

## Critério de Aceitação

- [ ] É possível abrir múltiplas cenas (.scene) em separadores diferentes.
- [ ] Cada separador renderiza apenas a sua própria hierarquia de entidades.
- [ ] Clicar num separador altera o foco do Inspector e Hierarchy para essa cena.
- [ ] Arrastar um separador para fora do editor cria uma janela flutuante (requer suporte a Viewports).
- [ ] Atalhos de teclado (Ctrl+Tab) permitem alternar entre cenas abertas.
- [ ] Fechar um separador com alterações não guardadas mostra um diálogo de confirmação.
