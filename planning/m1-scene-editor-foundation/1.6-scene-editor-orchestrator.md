## Visão Geral

O `SceneEditor` (Orchestrator) é o componente principal que une todas as janelas e ferramentas num único interface coerente. Ele é responsável por configurar o sistema de Docking do ImGui, gerir o menu principal da aplicação e garantir que o ciclo de vida de cada painel (Hierarchy, Inspector, Viewport, Asset Browser) seja respeitado.

Além do layout, o orquestrador gere as teclas de atalho (Shortcuts) globais, como `Ctrl+S` para guardar a cena ou `Ctrl+Z` para desfazer ações, delegando estas tarefas para os sistemas apropriados através do `EditorContext`.

## Implementação

### Layout de Docking

O editor utiliza o `ImGui::DockSpace` para permitir que o utilizador reorganize as janelas conforme a sua preferência.

```cpp
class SceneEditor {
    EditorContext m_context;
    
    // Painéis
    HierarchyPanel m_hierarchy;
    InspectorPanel m_inspector;
    SceneViewport m_viewport;
    AssetBrowser m_assetBrowser;
    
    // Ferramentas auxiliares existentes
    ConsoleWindow m_console;
    ProfilerWindow m_profiler;

public:
    void onImGuiRender() {
        renderMainMenuBar();
        setupDockspace();

        // Renderização de cada painel
        m_hierarchy.onImGuiRender();
        m_inspector.onImGuiRender();
        m_viewport.onImGuiRender();
        m_assetBrowser.onImGuiRender();
        m_console.onImGuiRender();
        m_profiler.onImGuiRender();
        
        renderStatusBar();
    }

private:
    void setupDockspace() {
        static bool opt_fullscreen = true;
        static ImGuiDockNodeFlags dockspace_flags = ImGuiDockNodeFlags_None;

        ImGuiWindowFlags window_flags = ImGuiWindowFlags_MenuBar | ImGuiWindowFlags_NoDocking;
        if (opt_fullscreen) {
            const ImGuiViewport* viewport = ImGui::GetMainViewport();
            ImGui::SetNextWindowPos(viewport->WorkPos);
            ImGui::SetNextWindowSize(viewport->WorkSize);
            ImGui::SetNextWindowViewport(viewport->ID);
            window_flags |= ImGuiWindowFlags_NoTitleBar | ImGuiWindowFlags_NoCollapse | ImGuiWindowFlags_NoResize | ImGuiWindowFlags_NoMove;
            window_flags |= ImGuiWindowFlags_NoBringToFrontOnFocus | ImGuiWindowFlags_NoNavFocus;
        }

        ImGui::Begin("DockSpace Demo", nullptr, window_flags);
        ImGuiID dockspace_id = ImGui::GetID("MyDockSpace");
        ImGui::DockSpace(dockspace_id, ImVec2(0.0f, 0.0f), dockspace_flags);
        ImGui::End();
    }

    void renderMainMenuBar() {
        if (ImGui::BeginMainMenuBar()) {
            if (ImGui::BeginMenu("File")) {
                if (ImGui::MenuItem("New Scene", "Ctrl+N")) { /* ... */ }
                if (ImGui::MenuItem("Open Scene...", "Ctrl+O")) { /* ... */ }
                if (ImGui::MenuItem("Save", "Ctrl+S")) { /* ... */ }
                ImGui::Separator();
                if (ImGui::MenuItem("Exit")) { /* ... */ }
                ImGui::EndMenu();
            }
            // ... Edit, View, Build menus
            ImGui::EndMainMenuBar();
        }
    }
};
```

### Gestão de Atalhos (Shortcuts)

```cpp
void handleShortcuts() {
    if (Input::isKeyDown(Key::Control)) {
        if (Input::isKeyPressed(Key::S)) saveCurrentScene();
        if (Input::isKeyPressed(Key::Z)) m_context.undoStack.undo(world);
        if (Input::isKeyPressed(Key::Y)) m_context.undoStack.redo(world);
    }
}
```

## Decisões de Design

| Decisão | Justificativa |
|---------|---------------|
| ImGui Docking Branch | Essencial para um IDE moderno que exige flexibilidade de janelas. |
| Orquestrador Singleton/Central | Simplifica a passagem de referências entre painéis, agindo como o "Dono" do `EditorContext`. |
| Barra de Estado (StatusBar) | Fornece feedback passivo crucial (como FPS e número de entidades) sem ocupar espaço útil. |

---

## Critério de Aceitação

- [ ] Sistema de Docking permite mover e agrupar todas as janelas.
- [ ] Barra de menus superior funcional (File, Edit, etc.).
- [ ] Atalhos de teclado (`Ctrl+S`, `Ctrl+Z`) executam as ações esperadas.
- [ ] O layout padrão é restaurado ao iniciar a aplicação (opcional para M1).
- [ ] A barra de estado mostra o caminho da cena atual e se existem alterações não guardadas.
- [ ] Janelas de consola e profiler existentes estão integradas no layout de docking.
